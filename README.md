# Gemini Plus

Gemini Plus is an Edge extension that hosts and prunes Google Gemini in a clean side panel for a more focused experience.

## Installation Guide

To install this extension in Microsoft Edge, follow these steps:

1.  **Download/Clone the Repository**:
    Ensure you have all the extension files (manifest.json, sidepanel.html, sidepanel.js, sidepanel.css, rules.json) in a single folder.

2.  **Open Edge Extensions Page**:
    Open Microsoft Edge and navigate to `edge://extensions`.

3.  **Enable Developer Mode**:
    In the bottom-left corner of the extensions page, toggle the **Developer mode** switch to **ON**.

4.  **Load Unpacked Extension**:
    - Click the **Load unpacked** button at the top of the page.
    - Navigate to the folder containing the extension files and click **Select Folder**.

5.  **Access the Side Panel**:
    - Click the **Gemini Plus** icon in your toolbar or the extensions menu.
    - The side panel will open on the right side of your browser.

## Features

- **Seamless Integration**: Ready-to-use with a single click.
- **Focused Interface**: Automatically hides the top banner and bottom hallucination disclaimer.
